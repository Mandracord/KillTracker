# KillTracker

An addon to track kills for Chaosbringer and Deathbringer to unlock DRK job. 

![demo](https://raw.githubusercontent.com/Mandracord/KillTracker/refs/heads/main/Demo.png)

## How it works

The addon tracks if you have Chaosbringer or Deathbringer equipped and kills a mob and tracks the kills for you. 

Do note - The addon do not track if you kill with other methods i.e magic. 

For the quest to progress you need to physically killed the enemy and cannot be done in group.

## Commands

```text 
//kt or //killtracker:

//kt hide | show - Hide or display the HUD
//kt reset - Resets the counter to 0. 
```
